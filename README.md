# PlaylistNerd

Nesse repositorio irei criar uma nova aplicação web que terá como objetivo criar playlist de um anime (opening, ending, trilhas sonoras) especifico que foi enviado.

Esse projeto consiste nas seguintes etapas:

1 - Fazer um Web Crawler que execute uma lista de animes e traga os 10 primeiros sites, assim irei verificar os melhores para buscar a a playlist

2 - Ao realizar uma consulta do animes, verificar entre os sites o nome das musicas e trazer todas as músicas únicas

3 - Criar Back-end em Laravel utilizando API

4 - Criar Front-end em Vue, consumindo as API desenvolvidas na etapa 3

5- Se tudo der certo, está pronto
